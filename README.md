# ⚡ SHQ-Tool v1.9 - All-in-One OSINT & Pentest Suite

SHQ-Tool, siber güvenlik araştırmacıları ve sızma testi uzmanları için geliştirilmiş, Termux ve Linux tabanlı sistemlerde çalışan çok fonksiyonlu bir istihbarat aracıdır.

## 🚀 Özellikler
* **Native OSINT Engine:** Kullanıcı adından 10+ popüler platformda otomatik hesap taraması (Sherlock bağımsız).
* **Network Analysis:** Nmap entegrasyonu ile hızlı ve agresif port tarama.
* **IP Intelligence:** IP adresi üzerinden konum ve ISP sorgulama.
* **Email Hunting:** Holehe motoru ile e-posta adresi kayıt kontrolü.
* **Web Security:** HTTP Header analizi ve Phishing (Oltalama) link kontrolü.
